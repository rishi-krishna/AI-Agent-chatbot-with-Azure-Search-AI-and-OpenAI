# Navigation and Settings

## Main navigation

- **Top menu**: Dashboard, Approvals, Reports, and your profile (top-right).
- **Left sidebar**: Shortcuts to Approvals, Reports, and other modules. Click the menu icon to collapse or expand.
- **Breadcrumbs**: Under the header; use them to go back to a parent section.

## Settings

- **Where**: Click your name or avatar (top-right) → **Settings**.
- **Profile**: Update name, email, and notification preferences under **Profile**.
- **Dashboard preferences**: Choose which widgets appear on the Dashboard.
- **Security**: Change password and manage sessions under **Security**.

## Screen guide

| What you need | Where to go |
|---------------|-------------|
| Approve requests | **Approvals** (main menu or sidebar) |
| Run or export reports | **Reports** → **Report Library** |
| View metrics | **Dashboard** |
| Change profile or preferences | **Settings** (top-right) |
| Scheduled reports | **Reports** → **Scheduled** |

Use the in-app chat if you’re not sure which screen to use; it can point you to the right place.
