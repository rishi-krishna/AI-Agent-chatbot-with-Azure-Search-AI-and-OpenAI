# Reports and Dashboard

## Dashboard

The **Dashboard** gives you a quick view of key metrics and actions.

- **Navigation**: Home → **Dashboard**, or **Dashboard** in the top menu.
- You’ll see widgets for approvals count, recent activity, and shortcuts to common tasks.

To change what appears, go to **Settings** → **Dashboard preferences** and choose which widgets to show.

## Reports

Reports let you run and export data for analysis.

- **Navigation**: Main menu → **Reports** → **Report Library**.
- **Report Library** lists all reports you can run. Click a report to open it.
- Use **Run** to refresh data and **Export** to download (e.g. Excel, PDF).

## Finding the right report

- In **Report Library**, use the search box or category filters.
- Each report has a short description; open it to see parameters and schedule options.
- For scheduled reports, go to **Reports** → **Scheduled** to view and manage them.

**Where do I see my scheduled reports?**  
Go to **Reports** → **Scheduled**. You can edit or delete a schedule from that screen.
