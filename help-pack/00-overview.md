# COO Application Overview

The COO (Chief Operating Office) application helps you manage operations, approvals, and reporting in one place.

## Main areas

- **Dashboard** – High-level metrics and quick actions. Open from the home screen or **Dashboard** in the top menu.
- **Approvals** – Review and approve requests. Go to **Approvals** in the main menu.
- **Reports** – Run and export reports. Use **Reports** → **Report Library**.
- **Settings** – Configure your profile and preferences under **Settings** (top-right).

## Getting help

Use the **Help** icon (?) in the header or the in-app chat to get answers and be guided to the right screen.
